## 安装运行
```git
$ git clone https://github.com/Binaryify/NeteaseCloudMusicApi
$ cd NeteaseCloudMusicApi
$ npm install
$ node app.js // 运行接口服务器 运行成功才能拿到数据
$ git clone git@github.com:chen-ziwen/chiko_music.git
$ cd CHIKO_MUSIC
$ npm install
$ npm run dev
```
